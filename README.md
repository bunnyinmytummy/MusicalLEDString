Musical LED String
==================

Addressable LED String Sample

### Requirement ###

* Arduino or variants
* Addressable LED String (WS2811 recommended)
* FastSPI Library https://code.google.com/p/fastspi/
* ArduinoFFT Library http://wiki.openmusiclabs.com/wiki/ArduinoFFT
